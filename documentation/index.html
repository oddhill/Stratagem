    fade 

    @mixin fade($transition-time: 0.3s, $delay: 0s, $timing: $easeInOut, $iteration-count: 1, $direction: normal) { ... }

    Description

    Gives an object a fade effect.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $transition-time

    Time the animation should take in seconds.

    		 Transition-time 0.3s
    $delay

    Delay given to a animation.

    		 Delay 0s
    $timing

    What kind of easing.

    		 Timing $easeInOut
    $iteration-count

    How many times the animations should iterate.

    		 Iteration-count 1
    $direction

    If the animation should be done once, do it backwards, or both at the same time.

    		 Direction normal

    Author

    • Anton Ranestam
