    v.2.0.0

    Make the web even nicer! :)

    Stratagem is a lightweight animation library built in Scss. No more bloated CSS files with duplicate keyframes and headaches importing keyframes. Just @include as usual and Stratagem takes care of the rest.

    Examples

    Usage & Installation

    1 $ bower install Stratagem

    Import Stratagem in the beginning of your base import file, all other stylesheets must be imported after.

    2 @import ‘stratagem’
    @import ‘modules/**/*’
    @import ‘content/**/*’

    All done
