    How to use

    Parameters

    Name Description Type Default value
    $transition-time Time the animation should take in seconds. Transition-time .3s
    $delay Delay given to a animation. Delay 0s
    $timing What kind of easing. Timing $easeInOut
    $iteration-count How many times the animations should iterate. Iteration-count 1
    $direction If the animation should be done once, do it backwards, or both at the same time. Direction normal
    $openClass Class to trigger animation. CSS class .open
    $color The color of the icon. Color #fff
